HTML

The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

Developed by: WHATWG Initial release: 1993; 29 years ago Latest release: Living Standard; 2022 Extended from: SGML Extended to: XHTML Contained by: Web browser

CSS

Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

Developed by: World Wide Web Consortium (W3C) Initial release: December 17, 1996; 25 years ago Latest release: CSS 2.1: Level 2 Revision 1; April 12, 2016; 5 years ago Container for: Style rules for HTML elements (tags)

BOOTSTRAP

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Developer(s): Bootstrap Core Team License: MIT License (Apache License 2.0 prior to 3.1.0) Repository: Bootstrap Repository Written in: HTML, CSS, Less (v3), Sass (v4) and JavaScript Stable release: 5.1.3 / 9 October 2021; 4 months ago Original author(s): Mark Otto, Jacob Thornton Initial release: August 19, 2011; 10 years ago

JavaScript, often abbreviated JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. Over 97% of websites use JavaScript on the client side for web page behavior, often incorporating third-party libraries.

Designed by: Brendan Eich of Netscape initially; others have also contributed to the ECMAScript standard First appeared: December 4, 1995; 26 years ago Paradigm: Multi-paradigm: event-driven, functional, imperative, object-oriented programming Typing discipline: Dynamic, weak, duck Stable release: ECMAScript 2021 / June 2021; 8 months ago

SQL

SQL is a domain-specific language used in programming and designed for managing data held in a relational database management system, or for stream processing in a relational data stream management system.

Developer: ISO/IEC Paradigm: Declarative Family: Query language Typing discipline: Static, strong Stable release: SQL:2016 / December 2016; 5 years ago Designed by: Donald D. Chamberlin; Raymond F. Boyce First appeared: 1974; 48 years ago
